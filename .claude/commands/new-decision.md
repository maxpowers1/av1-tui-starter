Create a new Architecture Decision Record:

1. Read the existing ADRs in `docs/adr/` to determine the next number
2. Use the template from `docs/adr/TEMPLATE.md`
3. Ask me for: the decision title, the options considered, and which I chose (or help me think through it)
4. Fill in the template with today's date
5. Set "Decided by" based on who drove the decision in this conversation
6. Add a row to `docs/decisions/DECISION_LOG.md`
7. Stage the files and draft a commit message using the convention in `docs/decisions/COMMIT_CONVENTION.md`
8. Show me the commit message before committing — I want to review the `decision:` trailer
